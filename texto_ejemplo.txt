Servicios:

- Autenticación de usuarios

Dependencias principales:
- Flask: Framework web para crear la API.
- SQLAlchemy: ORM para interactuar con la base de datos.
- Flask-Login: Extensión para manejar la autenticación de usuarios.
- Flask-Mail: Extensión para enviar correos electrónicos.
- psycopg2: Adaptador de base de datos para PostgreSQL.
- Jinja2: Motor de plantillas para generar HTML.
